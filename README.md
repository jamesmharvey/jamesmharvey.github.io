# numbersandotherthings.com

Source for [numbersandotherthings.com](https://numbersandotherthings.com) — built
with [Jekyll](https://jekyllrb.com/) (minima theme) and published via GitHub Pages.

- Posts live in `_posts/` as `YYYY-MM-DD-title.md`.
- Local preview: `bundle install && bundle exec jekyll serve`.
- Published automatically from the `main` branch.
